# Legal-Document-Review-System

AI-powered Legal Document Review System using NLP and Machine Learning for automated contract analysis

It helps in extracting key insights, summarizing content, identifying risks, and improving legal clarity.



Features

* Document content extraction and overview
* Key clause and risk analysis
* Legal terminology and clarity checking
* Contract classification and recommendation
* Automated document summary generation



Setup Instructions

1\. Install Python

Make sure Python 3.12 is installed and added to PATH.



2\. Install dependencies

Open PowerShell in your project directory:

cd "C:\\Users\\Alka\\Documents\\LegalRS\\legal-document-review-system"

pip install -r requirements.txt



3\. Run the application

python app.py
